## Selected Projects

### [AI Inventory Management System](https://github.com/iamkarandeepsingh/Inventory-Management-System)

An AI-assisted inventory analytics dashboard where users can ask natural-language questions, generate governed SQL plans, view KPIs, export results, and inspect explainable outputs.

**Built with:** FastAPI, DuckDB, Pandas, React, TypeScript, Tailwind CSS, Chart.js, Gemini API, JWT auth, Pytest

### [LLM & Transformer Notebooks](https://github.com/iamkarandeepsingh/Large-Language-Models-LLMs-)

A hands-on notebook collection exploring GPT-style text generation, Hugging Face Transformers, BERT question answering, and XLNet text classification.

**Focus:** prompt-conditioned generation, tokenization, hidden states, extractive QA, transformer inference workflows

### [AI Flight Search System](https://github.com/iamkarandeepsingh/Vibe-Coding-)

A natural-language flight search app using a multi-agent flow: one agent parses travel intent into structured JSON, and another queries live flight data.

**Built with:** Python, Flask, Gemini 2.5 Flash, Duffel API, multi-agent architecture

### [Phishing Website Detection](https://github.com/iamkarandeepsingh/Phishing-Website-Detection-using-Machine-Learning)

A machine learning system for classifying URLs as legitimate, suspicious, or phishing using multiple classifiers.

**Models:** Decision Tree, Random Forest, KNN, Neural Network

### [Weather Forecast Analysis](https://github.com/iamkarandeepsingh/Weather-Forecast-Analysis)

A data analysis workflow for cleaning, sampling, undersampling, and preparing weather datasets for exploratory analysis and modeling.

**Built with:** Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook
