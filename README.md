## Featured Projects

### AI Inventory Management System

An AI-assisted inventory analytics dashboard that lets users ask natural-language questions, generate governed SQL plans, view KPIs, export results, and inspect explainable outputs.

**Tech:** FastAPI, DuckDB, Pandas, React, TypeScript, Tailwind CSS, Chart.js, Gemini API, JWT auth, Pytest  
**Repo:** [Inventory-Management-System](https://github.com/iamkarandeepsingh/Inventory-Management-System)

### LLM & Transformer Notebooks

A hands-on collection of notebooks exploring GPT-style generation, Hugging Face Transformers, BERT question answering, and XLNet text classification.

**Focus:** prompt-conditioned generation, tokenization, hidden states, extractive QA, transformer inference workflows  
**Repo:** [Large-Language-Models-LLMs-](https://github.com/iamkarandeepsingh/Large-Language-Models-LLMs-)

### AI Flight Search System

A natural-language flight search application using a multi-agent flow: one agent parses user intent into structured JSON, another queries live flight data and returns usable results.

**Tech:** Python, Flask, Gemini 2.5 Flash, Duffel API, multi-agent architecture  
**Repo:** [Vibe-Coding-](https://github.com/iamkarandeepsingh/Vibe-Coding-)

### Phishing Website Detection

**Models:** Decision Tree, Random Forest, KNN, Neural Network  
**Repo:** [Phishing-Website-Detection-using-Machine-Learning](https://github.com/iamkarandeepsingh/Phishing-Website-Detection-using-Machine-Learning)

### Weather Forecast Analysis

A data analysis workflow for cleaning, sampling, undersampling, and preparing weather datasets for analysis and modeling.

**Tech:** Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook
